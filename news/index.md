# Changelog

## syntheticdata 0.1.1

- Initial release.
- [`synthesize()`](https://cuiweig.github.io/syntheticdata/reference/synthesize.md):
  Parametric (Gaussian copula), bootstrap with noise injection, and
  Laplace noise perturbation methods.
- [`validate_synthetic()`](https://cuiweig.github.io/syntheticdata/reference/validate_synthetic.md):
  Distributional similarity (KS), correlation preservation,
  discriminative accuracy (AUC), and nearest-neighbor privacy ratio.
- [`compare_methods()`](https://cuiweig.github.io/syntheticdata/reference/compare_methods.md):
  Benchmark all three synthesis methods on the same dataset.
- [`privacy_risk()`](https://cuiweig.github.io/syntheticdata/reference/privacy_risk.md):
  Re-identification risk assessment via nearest-neighbor distance ratio,
  membership inference, and attribute disclosure.
- [`model_fidelity()`](https://cuiweig.github.io/syntheticdata/reference/model_fidelity.md):
  Train-on-synthetic, test-on-real predictive performance comparison.
